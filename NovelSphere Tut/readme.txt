ノベルスフィア SDKには、以下のものが含まれています。

・novelsphere.js
・Builder
　グラフィカルな画面から novelsphere.js コンテンツのビルドを行うことが可能です。

##### 各フォルダの説明 #####

【LICENSE】
ノベルスフィア SDK に含まれるソフトウェアのライセンス情報が入っています。

【builder】
novelsphere.js コンテンツをビルドする際に必要なアプリケーションである Builder が入っています。
Builder の使い方は builder.txt を参照してください。

【plugin】
novelsphere.js 用のプラグイン群が入っています。
プラグインはこのフォルダに入れておくだけでは利用できません。
プラグインの利用法についてはドキュメントを参照してください。

【template】
novelsphere.js プロジェクトフォルダのテンプレートが入っています。
このフォルダをコピーして、プロジェクトフォルダの雛形としてお使いください。
タイトル画面やメニュー画面などがまとまった「Simple UI」が同梱されています。
通常はこちらをお使いください。

【template-plain】
novelsphere.js プロジェクトフォルダのテンプレートが入っています。
こちらは「Simple UI」が入っていない、よりシンプルなものになっています。
このフォルダをコピーして、プロジェクトフォルダの雛形としてお使いください。

※ノベルスフィア SDK にはドキュメント（説明書）は同梱されていません。
　ドキュメントはノベルスフィアデベロッパープログラム上からご覧いただけます。

　- https://developer.novelsphere.jp/doc/o2doc2/
